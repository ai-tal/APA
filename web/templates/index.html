Antenna Pattern Analyzer

A MATLAB App Designer application for loading, processing, and visualizing antenna far-field radiation patterns from industry-standard simulation formats.

📂
Multi-Format I/O
Reads .fz, .ffd, .ffe, .ffs, .out, .cut from HFSS, GRASP, CST, FEKO, XGTD
📊
Rich Plots
Polar cuts, contour maps, 3-D spherical, filled polar, coverage masks
⚙️
Pattern Processing
Gain, EIRP, axial ratio, PLF, HPBW, polarization, rotation, combine
